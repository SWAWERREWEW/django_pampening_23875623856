Для установки виртуального окружения достаточно просто запустить файл ins.bat через проводник, но можно просто нажать
на этот файл два раза. Для активации виртуального окружения в PyCharm нажать
```File → Setting → Project → Python Interpreter → Add interpreter → Add local intepreter → Location``` (Нажать на
иконку папки и выбрать появившуюся папку .venv) → Select exists interpreter → OK

## запуск сервера
# cd pampening_5
# python manage.py runserver

## Можно написать команду в консоли для установки виртуального окружения со всеми зависимостями
# ins.bat

## Это сработает, если в ```setting → Tools → Terminal → Shell path:``` указан файл cmd.exe
